The "Post-Stroke Health Monitoring App"

Built a web app that a user can create a new journal entry with the following attributes: date, description, picture, picture of tongue, video of mobility, health routines of the day, and blood pressure (along with annotations related to it). Conditions table also helps the caretaker isolate which conditions need special care separate from the daily journal entrying. The purpose of this is to help the user see which days or what health routines are important to keep up with for a patient with a stroke. Used microsoft excel for the Database Schema and wireframes. Tools used in the development of the App is Ruby on Rails, Vue.js,Javascript, Bootstrap, HTML, and CSS. Also check the Github links for the code.
